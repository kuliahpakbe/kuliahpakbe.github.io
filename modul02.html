Filesystem Hierarchy Standard (FHS)

Linux menggunakan satu pohon hierarki (Single Root Tree). Klik pada ikon folder di bawah untuk memahami fungsi setiap direktori standar industri.

Text Editor CLI: Nano vs Vim

Di dunia server murni, Anda tidak memiliki GUI. Penguasaan editor teks berbasis terminal adalah keharusan mutlak.

Perbandingan Karakteristik Editor

GNU Nano (Modeless)

Dirancang untuk kesederhanaan. Beroperasi dengan gaya WYSIWYG (What You See Is What You Get). Kontrol menggunakan pintasan keyboard kombinasi Ctrl.

Ctrl+O (Simpan) Ctrl+X (Keluar)

Vim (Modal Editor)

Diciptakan untuk efisiensi radikal. Tombol di keyboard memiliki fungsi berbeda tergantung pada Mode yang sedang aktif.

Mode Default. Digunakan untuk navigasi dan memanipulasi teks.
Ketik 'i' untuk Insert. Ketik 'dd' hapus baris.
