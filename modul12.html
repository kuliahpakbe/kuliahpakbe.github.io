Proyek Integrasi LAMP Stack

Inilah puncak perjalanan akademis Anda. Saatnya menggabungkan manajemen paket, administrasi layanan, dan konfigurasi keamanan untuk membangun pondasi server web kelas industri yang siap menampung jutaan pengguna.

Mulai Rakit Server

Anatomi Tumpukan Layanan (The Stack)

Sebuah situs web dinamis (seperti SIAKAD atau E-Commerce) tidak bisa berjalan mandiri. Ia membutuhkan kolaborasi rumit dari 4 lapisan perangkat lunak yang saling berinteraksi, yang dikenal dengan akronim LAMP.

Linux

Sistem operasi dasar (Ubuntu Server). Sebagai fondasi keras yang menyediakan sistem berkas (file system), manajemen jaringan, dan alokasi memori.

Apache

Web Server (*daemon*). Bertugas mendengarkan ketukan di Port 80 (HTTP) dari browser pengguna, lalu mengantarkan berkas web (HTML/CSS) kembali ke pengguna.

MySQL

Database Server (*daemon* di Port 3306). Tempat penyimpanan persisten data terstruktur seperti akun pengguna, produk, dan transaksi.

PHP

Bahasa Pemrograman *Server-Side*. Otak yang memproses logika bisnis. Ia membaca dari Database (MySQL) lalu memberikan hasilnya ke Web Server (Apache).

Uncomplicated Firewall (UFW)

Setelah Anda menginstal LAMP Stack, port server Anda (seperti 3306 milik database) berisiko terekspos ke seluruh dunia. Sistem operasi harus digembok menggunakan Tembok Api (Firewall).

ATURAN EMAS FIREWALL: "Tolak semuanya secara default (Deny All), lalu buka HANYA lubang-lubang spesifik yang benar-benar dibutuhkan."

  • sudo ufw enable : Menyalakan Tembok Api.
  • sudo ufw allow 22 : Membuka akses Remote SSH (Sangat Kritis, jika lupa dibuka Anda akan terkunci dari server!).
  • sudo ufw allow 80 : Membuka akses lalu lintas web HTTP agar pengunjung bisa melihat situs Anda.
